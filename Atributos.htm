No. Atributo Etiqueta HTML5 Descripción
1 accept <form>, <input> Tipos de archivos aceptados por el servidor. Puedes definirlos con valores separados por comas, por ejemplo: .doc, .ods, .txt, etc. O también puedes indicarle un conjunto de tipos de archivos, como por ejemplo: «video/*» o «audio/*»
2 accept-charset <form> Define una lista de caracteres soportados en el formulario. Por ejemplo: accept-charset=»ISO-8859-1″
3 accesskey Atributo Global Podrás configurar un botón de tu teclado para que se ejecute acción en la página web. Por ejemplo, un enlace hacia la página de contacto o la home.
4 action <form> El atributo especifica la URL dónde se enviarán los datos recogidos en el formulario, a menudo un archivo .php dentro del mismo servidor.
5 align <applet>, <caption>, <col>, <colgroup>, <hr>, <iframe>, <img>, <table>, <tbody>, <td>, <tfoot> , <th>, <thead>, <tr> Especifica el alineamiento horizontal del elemento.
Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar.
6 alt <applet>, <area>, <img>, <input> Texto alternativo que se mostrará en el navegador en caso de que la imagen no se pueda mostrar. Muy importante de cara a accesibilidad.
7 async <script> Indica que el script debe ejecutarse de forma asíncrona.
8 autocomplete <form>, <input> Indica si los controles en este formulario pueden tener por defecto sus valores completados automáticamente por el navegador.
9 autofocus <button>, <input>, <keygen>, <select>, <textarea> El elemento debe tener el foco automáticamente una vez se cargue la página web.
10 autoplay <audio>, <video> Hace que el audio o video comience a reproducirse una vez se cargue la página web.
11 bgcolor <body>, <col>, <colgroup>, <marquee>, <table>, <tbody>, <tfoot>, <td>, <th>, <tr> Te permite añadir un color de fondo al elemento HTML. El color debe ser añadido en el modo de color RGB, por ejemplo #FF00AA.
Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar.
12 border <img>, <object>, <table> Permite definir un borde al elemento HTML.
Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar.
13 buffered <audio>, <video> Con este atributo podrás definir un rango de tiempo del elemento multimedia que será usado como buffer.
14 charset <meta>, <script> Atributo necesario par definir la codificación de caracteres de la página o del script.
15 checked <command>, <input> Usado para definir que un elemento HTML debe estar marcado por defecto al cargar la página web.
16 cite <blockquote>, <del>, <ins>, <q> Contiene una URL que apunta a la fuente de la cita o cambio. Muy importante de cara a una correcta optimización SEO de la página.
17 class Atributo Global Otorga una clase creada en CSS a un elemento HTML. Se usa principalmente para aplicar estilos a elementos con propiedades en común.
18 code <applet> Especifica la URL del archivo de tipo applet que será cargado y ejecutado.
19 color <basefont>, <font>, <hr> Atributo usado para indicar el color del texto de un elemento.
Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar.
20 cols <textarea> Dentro de una etiqueta <textarea>, se usa para definir el número de columnas que tendrá el campo de texto.
21 colspan <td>, <th> Dentro de una tabla, define el número de columnas que una celda debe ocupar.
22 content <meta> Un atributo que permite definir el contenido de la etiqueta HTML <meta> que acompaña.
23 contenteditable Atributo Global Define si el contenido del elemento HTML podrá ser editable o no.
24 contextmenu Atributo Global Define el ID del elemento <menu> que servira como menú de contexto para otro elemento.
25 controls <audio>, <video> Indica si el navegador debe mostrar los controles multimedia del reproductor integrado al usuario.
26 coords <area> Un conjunto de valores que especifican las coordenadas dentro de la etiqueta area.
27 data <object> Especifica la URL, absoluta o relativa del objeto o recurso.
28 data-* Atributo Global Permite asociar atributos presonalizados a un elemento HTML.
29 datetime <del>, <ins>, <time> Permite definir la fecha y hora asociadas con el elemento.
30 default <track> Con este atributo podrás definir que por defecto, una pista por ejemplo de subtítulos esté habilitada. Siempre y cuando las preferencias del usuario no indiquen algo diferente.
31 defer <script> Indica que el código debe ser ejecutado despues de que la página este cargada.
32 dir Atributo Global Con este atributo puedes definir la dirección del texto en un elemento HTML. Los valores permitidos son ltr (Izquierda-a-Derecha) o rtl (Derecha-a-Izquierda).
33 dirname <input>, <textarea> Similar a la anterior. Indica la dirección del texto asociada con el elemento al momento de enviar el formulario.
34 disabled <button>, <command>, <fieldset>, <input>, <keygen>, <optgroup>, <option>, <select>, <textarea> Permite que un elemento HTML esté deshabilitado y el usuario no pueda interactuar con el.
35 download <a>, <area> Si este atributo se añade al elemento HTML, el recurso indicado podrá descargarse al dispositivo del usuario.
36 draggable Atributo Global Define si el elemento puede ser arrastrado.
37 enctype <form> Define el tipo de contenido del formulario cuando el método del mismo es POST.
38 for <label>, <output> Atributo usado para asociar o describir elementos que pertenecen a éste.
39 form <button>, <fieldset>, <input>, <keygen>, <label>, <meter>, <object>, <output>, <progress>, <select>, <textarea> Atributo para indicar el formulario al que pertenece el elemento.
40 headers <td>, <th> A través de «ids» indicados en los encabezados de una tabla <th> podrás asociarlos a diferentes elementos de la tabla <td>. Muy útil de cara a la accesibilidad y por lo tanto para el SEO.
41 height <canvas>, <embed>, <iframe>, <img>, <input>, <object>, <video> Permite especificar la altura de los elementos HTML indicados.
Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar.
42 hidden Atributo Global Este atributo permite indicar que el elemento HTML no debe ser visible. Es habitual usarlo por ejemplo, para ocultar elementos que no deben ser visibles inicialmente pero que después de una acción si deben verse.
43 high <meter> El atributo especifica el intervalo donde el atributo «value» tomara el valor como alto.
44 href <a>, <area>, <base>, <link> La URL de un recurso asociado.
45 hreflang <a>, <area>, <link> Especifica el lenguaje del recurso asociado.
46 http-equiv <meta> Indica una directiva que puede alterar el comportamiento de la página y como es leída por los exploradores.
47 id Atributo Global Usado para asignar un identificador único a un elemento HTML. Usado generalmente para acceder al mismo desde CSS o Javascript. No pueden existir dos «ids» iguales en una misma página web.
48 ismap <img> Indica que la imagen es parte de un mapa de imagen del servidor.
49 itemprop Atributo Global Atributo que permite dar más información a Google acerca de imágenes o datos dentro de cualquier tipo de etiquetas. Por ejemplo se puede indicar propiedades como: actores , clasificación, género, etc .
50 label <track> Especifica el título de la pista con un formato legible por el usuario. Usado conjuntamente con el atributo «for»
51 lang Atributo Global Define el lenguaje utilizado en el elemento.
52 language <script> Define el lenguaje (de programación) utilizado en el script.
53 list <input> Identifica una serie de valores predefinidos para ser sugeridos al usuario en un menú desplegable.
54 loop <audio>, <bgsound>, <marquee>, <video> Atributo que habilita al medio a volver a reproducirse desde el principio cuando llega al final.
55 low <meter> Similar al atributo high, solo que en este caso, permite definir cuando un valor estará en el rango bajo.
56 max <input>, <meter>, <progress> Define el máximo valor aceptado.
57 maxlength <input>, <textarea> Define el mayor número de caracteres aceptados.
58 media <a>, <area>, <link>, <source>, <style> Indica el tipo de medio para el cual el recurso vinculado fue creado.
59 method <form> Define el método HTTP a emplear para enviar el formulario. Puede ser GET (predeterminado) o POST.
60 min <input>, <meter> Define el valor mínimo aceptado.
61 multiple <input>, <select> Atributo que define si se puede introducir múltiples valores.
62 muted <video> Al reproducir un contenido audiovisual, este atributo indica si el audio será silenciado inicialmente al cargar la página.
63 name <button>, <form>, <fieldset>, <iframe>, <input>, <keygen>, <object>, <output>, <select>, <textarea>, <map>, <meta>, <param> Nombre del elemento. Utilizado a menudo por el servidor para identificar los campos en el envío de formularios.
64 novalidate <form> Con este atributo indicamos al formulario que no valide el contenido del mismo.
65 open <details> Indica si los detalles de la página seran mostrados cuando cargue la misma.
66 optimum <meter> Con este atributo se define el valor numérico óptimo.
67 pattern <input> Este atributo define una expresión regular con la cual el valor del elemento debe ser validado antes de aceptar el contenido del mismo.
68 ping <a>, <area> Indica si se enviara un ping o notificacion a la URL del recurso cuando el usuario utlize el vinculo.
69 placeholder <input>, <textarea> Proporciona una sugerencia al usuario de lo que puede ser introducido en el campo de un formulario por ejemplo. Muy util de cara a la accesibilidad.
70 poster <video> Atributo que permite añadir una imagen que se mostrará mientras se carga un video en una web. En caso de que la URL del video este «rota», el poster seguirá cargándose consiguiendo con esto no ver un hueco vacio en la web.
71 preload <audio>, <video> Indica si el recurso o partes del mismo deben ser precargadas.
72 readonly <input>, <textarea> Con este atributo se define si el elemento puede o no ser editado por el usuario.
73 rel <a>, <area>, <link> Especifica la relación entre el objeto destino y el objeto enlace. Importante de cara una correcta optimización SEO.
74 required <input>, <select>, <textarea> Con este atributo hacemos que la etiqueta Html sea, o no, de obligada cumplimentación.
75 reversed <ol> Con este atributo podemos modificar el comportamiento estándar de una lista ordenada haciendo que se vea listada en orden descendente en lugar de ascendente.
76 rows <textarea> Define el número de filas en un área de texto.
77 rowspan <td>, <th> Define el número de filas que una celda de una tabla debe abarcar.
78 sandbox <iframe> Lista de restricciones a ser desactivadas en el iframe.
79 scope <th> En una tabla permite definir las celdas que la cabecera deberá ocupar en este elemento.
80 seamless <iframe> Indica si el iframe debe ser cargado como parte del mismo documento.
81 selected <option> Define un valor que aparecerá por defecto seleccionado al cargar la página.
82 size <input>, <select> Define el ancho del elemento (en píxeles). En este caso es mejor por tanto usar reglas CSS. En cambio. si el atributo del elemento es del tipo «text» o «password», entonces es el número de caracteres que se podrán introducir.
83 span <col>, <colgroup> Indica el número de columnas que se agrupan en una tabla.
84 spellcheck Atributo Global Con este atributo se permite que un texto pueda ser autocorregido ortográficamente, o no.
85 src <audio>, <embed>, <iframe>, <img>, <input>, <script>, <source>, <track>, <video> La URL del contenido integrable en la etiqueta HTML que lo acompaña.
86 srcdoc <iframe> Especifica el contenido de la página incluida.
87 srclang <track> Especifica el lenguaje del contenido de la pista.
88 srcset <img> Este atributo define diferentes versiones de la imagen para mostrar una o otra según el tipo de pantalla.
89 start <ol> Define el número inicial de la lista ordenada (si es diferente a 1).
90 style Atributo Global Define los estilos CSS que anulan los estilos establecidos previamente. En este caso son llamados estilos en línea ya que se introducen directamente en la etiqueta HTML en cuestión.
91 target <a>, <area>, <base>, <form> Con este atributo se determina el marco destino en un vínculo.
Valores especiales: «_blank» indica una nueva ventana, «_parent» indica el marco padre del marco que contiene el código fuente, «_self» indica el marco donde está el código fuente y «_top» indica la ventana completa del navegador.
92 title Atributo Global Texto a ser mostrado cuando el cursor esté sobre el elemento. Muy útil de cara a la usabilidad y para una correcto posicionamiento SEO.
93 type <button>, <input>, <command>, <embed>, <object>, <script>, <source>, <style>, <menu> Define el tipo de elemento HTML.
94 usemap <img>, <input>, <object> Indica la URL parcial de un image map asociado con el elemento.
Note: Este atributo no puede ser usado si el elemento <img> desciende de un <button> o <a>
95 value <button>, <option>, <input>, <li>, <meter>, <progress>, <param> Define el valor predeterminado a ser mostrado al cargar la página.
96 width <canvas>, <embed>, <iframe>, <img>, <input>, <object>, <video> Para los elementos enumerados aquí, esto establece el ancho del elemento. Aunque como en caso anteriores, lo correcto sería usar reglas CSS.
97 wrap <textarea> Indica la forma en la cual el texto debe ser envuelto.
